# Antique Blog

Welcome to the Antique Blog project! This blog is dedicated to exploring the world of reverse engineering software on the Commodore 64 (C64). Here, we share insights, tutorials, and discussions about retro computing and software analysis.

## Project Structure

- **content/**: Contains all blog posts and drafts.
  - **posts/**: Published blog posts in Markdown format.
  - **drafts/**: Unpublished drafts for future posts.
  
- **themes/**: Contains the blog's themes and layouts.
  - **default/**: The default theme for the blog.
    - **layouts/**: HTML layouts for posts and the index page.
    - **assets/**: Additional assets like images and fonts.

- **static/**: Static files for the blog.
  - **css/**: Stylesheets for the blog.
  - **js/**: JavaScript files for interactivity.

- **scripts/**: Contains scripts for building or deploying the blog.
  
- **package.json**: Configuration file for npm dependencies and scripts.

- **config.yml**: Configuration settings for the blog.

- **.gitignore**: Specifies files to be ignored by Git.

## Getting Started

To get started with the Antique Blog, clone the repository and install the necessary dependencies:

```bash
git clone <repository-url>
cd antique-blog
npm install
```

## Usage

To build or serve the blog, run the following command:

```bash
npm run build
```

or

```bash
npm start
```
